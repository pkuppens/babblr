# Trademarks and Branding

This repository is open source under AGPL-3.0, but project names and branding can still be protected.

## Babblr name and logo

"Babblr" and related branding may be trademarks of the project maintainers.
This document describes intended usage guidelines.

## Allowed (generally OK)

- Referring to the project by name in a factual way (for example, "Compatible with Babblr")
- Using the name in articles, reviews, and tutorials
- Forks that clearly indicate they are forks (for example, "Babblr (community fork)")

## Not allowed without permission

- Using "Babblr" as the name of a derived product in a way that suggests it is the official product
- Using Babblr logos/branding in marketing that implies endorsement
- Registering confusingly similar domains or app store listings

## Requests

If you want to use the Babblr name or branding in a commercial context, please open an issue and tag `@pkuppens`.
