# Glossary

## Purpose

Consistent terminology used throughout Babblr PRDs.

## General Terms

**Babblr**: The desktop language learning application

**AI Tutor**: The conversational AI assistant (Claude, Gemini, Ollama)

**Target Language**: The language the student is learning

**Native Language**: The student's first language

## Language Learning Terms

**CEFR**: Common European Framework of Reference for Languages (A1-C2)

**Proficiency Level**: Student's current CEFR level

**Correction**: Grammar or vocabulary error feedback

**Mastery Score**: Measure of vocabulary retention

**Comprehensible Input**: Language slightly above current level (i+1)

## Technical Terms

**LLM Provider**: AI service (Claude, Gemini, Ollama)

**STT**: Speech-to-Text (Whisper)

**TTS**: Text-to-Speech (Edge TTS)

**Dual Licensing**: AGPL-3.0 + Commercial license

**CLA**: Contributor License Agreement

**BYOAPI**: Bring Your Own API (user provides API keys)
