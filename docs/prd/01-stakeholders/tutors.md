# Tutor Stakeholders

## Purpose

This document defines tutor personas, needs, and how Babblr may serve tutors in the future. Tutors are currently **NOT** a primary stakeholder but represent a future expansion opportunity.

## Current Status

**Tutor features are NOT currently implemented.** This document outlines future vision (Priority 3).

## Tutor Personas (Future)

### Persona 1: Independent Language Tutor

**Demographics**:
- Age: 28-45
- Occupation: Freelance language tutor
- Location: Global
- Students: 10-30 active students

**Potential Needs**:
- Insights into student progress and common mistakes
- Tools to create custom conversation topics for students
- Analytics dashboards showing student performance
- Integration with existing tutoring workflow

**How Babblr Could Help** (Future):
- Student analytics dashboard
- Custom topic creation tools
- Progress reports for students
- Supplemental practice between lessons

---

### Persona 2: Language School Teacher

**Demographics**:
- Age: 30-50
- Occupation: Language teacher at institution
- Students: 50-200 per semester

**Potential Needs**:
- Scalable conversation practice for students
- Assessment tools aligned with curriculum
- Progress tracking across class
- Homework assignment and grading

**How Babblr Could Help** (Future):
- Classroom licenses
- Custom assessments
- Student progress dashboards
- LMS integration

---

## Validation Plan

Before implementing tutor features, we will:

1. **Research**: Interview 10+ tutors about needs and workflows
2. **Prototype**: Build MVP tutor dashboard
3. **Pilot**: Beta test with 5-10 tutors
4. **Iterate**: Refine based on feedback
5. **Launch**: Roll out to broader tutor community
