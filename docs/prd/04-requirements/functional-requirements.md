# Functional Requirements

## Purpose

Specifies what the system must do.

## FR-1: Conversation Management

**FR-1.1**: System shall allow users to create new conversations
**FR-1.2**: System shall save conversations to local database
**FR-1.3**: System shall allow resuming saved conversations
**FR-1.4**: System shall display conversation history

## FR-2: AI Conversation

**FR-2.1**: System shall integrate with LLM providers (Claude, Gemini, Ollama)
**FR-2.2**: System shall adapt conversation difficulty to CEFR level
**FR-2.3**: System shall provide error corrections with explanations

## FR-3: Speech Processing

**FR-3.1**: System shall transcribe speech using Whisper STT
**FR-3.2**: System shall generate speech using TTS

## FR-4: Vocabulary Tracking

**FR-4.1**: System shall extract vocabulary from conversations
**FR-4.2**: System shall track vocabulary mastery

## FR-5: Assessment System

**FR-5.1**: System shall provide CEFR assessments
**FR-5.2**: System shall recommend proficiency level
